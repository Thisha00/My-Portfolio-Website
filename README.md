# 🌐 Personal Portfolio Website

This is my personal portfolio website created using **HTML and CSS**.
The website presents information about me, my education, skills, experience, and projects.

The purpose of this portfolio is to showcase my work, technical skills, and learning journey in the field of Information Technology.

---

## 🚀 Features

* 🏠 **Home Section** – Introduction and welcome message
* 👩‍💻 **About Section** – Information about me and my interests
* 🎓 **Education Section** – My academic background
* 🧠 **Skills Section** – Technical skills and tools I use
* 💼 **Experience Section** – My learning experiences and activities
* 📂 **Projects Section** – Some projects I have developed

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in your web browser
